== Installation ==

This section describes how to install the plugin and get it working.

<h2>Automatically</h2>

1. Search for UltimateWoo WooCommerce Plugin in the Add New Plugin section of the WordPress admin
2. Install & Activate

<h2>Manually</h2>

1. Download the zip file and upload `ultimatewoo` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
